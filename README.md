#Rapido Ride Data Analysis (SQL Project)

# Project Overview
This project analyzes Rapido ride booking data using MySQL.
The goal is to perform data cleaning, exploratory data analysis (EDA),
and generate business insights using SQL queries.

The dataset contains ride-level information such as booking value,
distance, vehicle type, payment method, ratings, and cancellations.



# Dataset Information
-> Total Records: 200
-> Valid Records After Cleaning: 199
-> Completed Rides: 165
-> Cancelled Rides: 24
-> Incomplete Rides: 10
-> Cities Covered: Hyderabad, Delhi, Chennai, Pune, Bengaluru

Each row represents one ride transaction.

-> Hyderabad generated the highest revenue.
-> UPI is the most preferred payment method.
-> 83% ride completion rate indicates strong operations.
-> Highest single ride revenue: ₹246.90.
-> High revenue per KM suggests profitable short-distance rides.
